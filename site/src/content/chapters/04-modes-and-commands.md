Yabomish 內建命令系統，可切換輸入模式、執行工具操作、進入查詢功能。

---

## 4.1 命令系統

輸入兩個逗號 `,,` 進入命令模式，接著打命令碼，再按**空白鍵**執行。

```
,, + 命令碼 + Space
```

例如：輸入 `,,S` 再按空白鍵，即切換至簡體中文模式。

> **特殊用法**：`,,` 之後直接按空白鍵，會輸出一個**全型空格**（U+3000）。


## 4.2 輸入模式

| 命令 | 模式 | 說明 |
|------|------|------|
| `,,T` | 繁體中文 | 預設模式 |
| `,,S` | 簡體中文 | 只顯示字表中本身就是簡體的字 |
| `,,SP` | 速打 | 僅保留最短碼——同一字有多組編碼時，只留最短的 |
| `,,SL` | 慢打 | 僅保留最長碼 |
| `,,TS` | 繁→簡轉換 | 打繁體碼，輸出對應的簡體字 |
| `,,ST` | 簡→繁轉換 | 打簡體碼，輸出對應的繁體字 |
| `,,J` | 日文假名 | 查詢 `編碼+','` 與 `編碼+'.'` 對應的假名字表 |


### 繁體 vs. 繁→簡的差異

- `,,S`（簡體中文）：字表中原本就收錄為簡體的字才會出現
- `,,TS`（繁→簡轉換）：使用繁體編碼輸入，送字時自動透過 OpenCC 轉換為簡體

## 4.3 工具命令

| 命令 | 功能 |
|------|------|
| `,,RS` | 重置字頻統計——清除所有 unigram / bigram 學習資料 |
| `,,RL` | 重載字表與擴充表——修改擴充表後用此命令即時生效 |
| `,,PIN` | 固定同碼字排序——輸入碼後選字指定順序，不受 decay 影響 |
| `,,UNPINx` | 解除碼 x 的固定排序（如 `,,UNPINa`） |
| `,,C` | 顯示當前模式——螢幕中央彈出目前的輸入模式名稱 |
| `,,H` | 命令說明——顯示所有可用命令的快速參考 |

> **提示**：打錯命令時會顯示「未知命令」並建議輸入 `,,H` 查看說明。

## 4.4 查詢命令

以下命令會進入特殊查詢模式，詳細操作請參閱[第六章](/yabomish/chapters/06-lookup-modes)。

| 命令 | 功能 |
|------|------|
| `,,ZH` | 注音查碼——輸入注音符號，查詢對應的嘸蝦米編碼 |
| `,,PYS` | 拼音查碼（簡體）——輸入拼音字母＋聲調數字查碼 |
| `,,PYT` | 拼音查碼（繁體）——同上，但查繁體字 |
| `,,TO` | 同音字查詢——送字後列出所有同音字 |

## 4.5 模式提示

切換模式時，螢幕中央會短暫顯示 **toast 提示**，標示新模式名稱：

| 模式 | 提示文字 |
|------|----------|
| 繁體中文 | 繁中 |
| 簡體中文 | 簡中 |
| 英文 | A |
| 注音查碼 | 注 |
| 速打 | 速 |
| 日文假名 | 仮 |

### 切入提示

從其他輸入法切換到 Yabomish 時，也會顯示當前模式的 toast 提示，方便確認狀態。此功能可在設定程式中關閉。
