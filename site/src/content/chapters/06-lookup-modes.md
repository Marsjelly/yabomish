Yabomish 提供多種查碼模式，讓你用注音、拼音或同音字來反查嘸蝦米碼。這些模式適合學習新字的拆碼方式，或是遇到只知道讀音但不確定怎麼拆的字時使用。


---

## 同音字查詢

輸入 `,,TO` + **Space** 進入同音字模式，狀態列會顯示「同音」提示。

### 使用方式

1. 進入同音字模式後，照常用嘸蝦米打字
2. 每次送出一個字，選字窗會列出該字的所有同音字（依注音分組）
3. 用數字鍵選擇想要的同音字
4. 選字後，畫面會以 toast 顯示該字的嘸蝦米碼提示
5. 再次輸入 `,,TO` + **Space** 退出同音字模式

### 多音字設定

預設情況下，同音字查詢只顯示最常用讀音對應的同音字。例如查「同」時，預設只列出 ㄊㄨㄥˊ（同意）的同音字，不會列出 ㄊㄨㄥˋ（痛）的同音字。讀音順序依據萌典（教育部國語辭典）的標準排列，並以 wiki 語料庫頻率校正。

如果需要查看所有讀音（如「行」的 ㄏㄤˊ 和 ㄒㄧㄥˊ），可在設定程式的「輸入」頁中開啟：

> ☑️ 同音字顯示多音字罕見讀音

開啟後，同音字列表會包含該字所有讀音的同音字，以讀音分組顯示。

## 注音反查

輸入 `,,ZH` + **Space** 切換到注音反查模式。狀態列會顯示「注音」提示。


### 使用方式

1. 進入注音模式後，鍵盤切換為標準注音配置
2. 輸入注音符號組合，再按聲調鍵完成一個音節
3. 候選字以「**字　碼**」格式顯示，例如：

   ```
   1. 好  hao    2. 號  haw    3. 浩  hgk
   ```

4. 按數字鍵選字，選字後**自動退出**注音模式，回到嘸蝦米
5. 如果不選字，再次輸入 `,,ZH` + **Space** 可手動切回嘸蝦米

### 聲調鍵對照

| 按鍵      | 聲調   | 符號 | 範例       |
|----------|--------|------|-----------|
| **Space** | 一聲   | （無）| ㄏㄠ → 蒿  |
| **3**     | 三聲   | ˇ    | ㄏㄠˇ → 好 |
| **4**     | 四聲   | ˋ    | ㄏㄠˋ → 號 |
| **6**     | 二聲   | ˊ    | ㄏㄠˊ → 豪 |
| **7**     | 輕聲   | ˙    | ˙ㄇㄚ → 嗎 |

> 💡 聲調鍵的配置與標準注音輸入法相同（3、4、6、7 對應三聲、四聲、二聲、輕聲）。

## 拼音查碼

用漢語拼音來查嘸蝦米碼，適合熟悉拼音但不確定拆碼的使用者。

### 進入方式

| 命令                  | 模式                     |
|----------------------|--------------------------|
| `,,PYS` + **Space**  | 拼音查碼（簡體字＋簡體碼）  |
| `,,PYT` + **Space**  | 拼音查碼（繁體字＋繁體碼）  |

狀態列會顯示「拼音簡」或「拼音繁」提示。

### 使用方式

1. 輸入拼音字母（小寫英文），例如 `hao`
2. 按聲調數字鍵選擇聲調：

   | 按鍵      | 聲調   | 範例          |
   |----------|--------|--------------|
   | **Space** | 一聲   | hao → 蒿     |
   | **1**     | 一聲   | hao1 → 蒿    |
   | **2**     | 二聲   | hao2 → 豪    |
   | **3**     | 三聲   | hao3 → 好    |
   | **4**     | 四聲   | hao4 → 號    |
   | **5**     | 輕聲   | ma5 → 嗎     |

3. 候選字以「**字　碼**」格式顯示，例如：

   ```
   1. 好  hao    2. 郝  hkl    3. 昊  hgk
   ```

4. 按數字鍵選字

### 特殊拼音處理

- **v → ü**：輸入 `v` 自動轉換為 `ü`，例如輸入 `lv` 查詢 `lü`（綠）
- **再次輸入命令**（`,,PYS` 或 `,,PYT`）+ **Space** 退出拼音模式

## 拆碼提示

拆碼提示不是獨立的查碼模式，而是一個輔助功能。開啟後，每次送字都會以 toast 短暫顯示該字的嘸蝦米碼。

### 開啟方式

在設定程式的「輸入」頁中勾選：

> ☑️ 拆碼提示

開啟後，正常打字送出每個字時，畫面右上角會短暫顯示該字的拆碼，例如：

```
好 → hao（女子）
```

這個功能適合嘸蝦米初學者，幫助你在日常打字中逐漸記住每個字的拆碼方式。熟練後可以關閉。
