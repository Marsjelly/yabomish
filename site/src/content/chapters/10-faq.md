## 安裝

### Q：安裝後在輸入方式列表找不到 Yabomish？

登出再登入（或重新開機）。macOS 有時需要重新載入輸入法列表，才會顯示新安裝的輸入方式。

登入後前往：**系統設定 → 鍵盤 → 輸入方式 → 「+」**，在列表中找到「Yabomish」加入即可。

### Q：編譯失敗？

請確認：

1. 已安裝 Xcode Command Line Tools：
   ```bash
   xcode-select --install
   ```
2. macOS 版本為 **14.0（Sonoma）** 以上
3. 使用 **Apple Silicon**（M1/M2/M3/M4）Mac

## 使用

### Q：打字沒有候選字？

確認已匯入嘸蝦米字表。開啟設定程式（YabomishPrefs.app），前往 **輸入** 分頁 → **匯入字表⋯** → 選擇你的 `liu.cin` 檔案。

匯入後字表會在裝置上編譯為 `liu.bin`，過程約數秒。

### Q：聯想建議太多／太少？

前往設定程式 → **聯想與詞庫** 分頁：

- 調整各詞庫的開關（逐一啟用或停用）
- 拖拉調整詞庫優先順序
- 或直接關閉聯想輸入功能

### Q：想用台灣用詞，不想看到中國用詞？

設定程式 → **輸入** → **用詞習慣** → 選擇「**臺灣正體**」。

中國用詞不會完全消失，而是會被**降權**——排到候選列表的後面。這樣偶爾需要時仍然找得到。

### Q：候選字窗不見了？

可能發生在全螢幕 App 中。嘗試：

- 設定程式 → **輸入** → **選字窗模式** → 切換為「**固定位置**」模式
- 固定位置模式下，候選字窗會顯示在螢幕固定位置，不受 App 視窗影響

### Q：Chrome 網址列候選字窗位置不對？

此問題已在 **v0.3.44** 修正。請更新到最新版本。

如更新後仍有問題，可切換到固定位置模式作為替代方案。

### Q：某些 App 中候選字窗不顯示？

部分 GPU 終端（如 Ghostty）可能回傳無效的游標座標。Yabomish 偵測到無效座標時，會自動 fallback 到固定位置模式。

如果自動 fallback 未生效，請手動切換到固定位置模式。

## 資料

### Q：記憶體佔用多少？

約 **70MB**，但大部分是 mmap 虛擬記憶體。

作業系統會按需載入頁面，實際佔用的實體 RAM 遠小於此數字。這是正常現象，不需擔心。

### Q：字頻學習資料在哪？

```
~/Library/YabomishIM/freq.db
```

這是一個 SQLite 資料庫（WAL 模式）。如需重置字頻統計，輸入 `,,RS` + 空白鍵。

### Q：可以跨機同步嗎？

可以。設定程式 → **輸入** → **同步資料夾**，指定一個 iCloud Drive 路徑即可。

同步範圍包含字頻學習資料和擴充表。

### Q：liu.cin 會上傳嗎？

**不會**。字表在裝置上編譯為 `.bin`，完全離線處理，不會上傳到任何伺服器。

## 疑難排解

### Q：如何開啟 Debug 模式？

設定程式 → **外觀** → 開啟 **Debug 模式**。

日誌會寫入：

```
~/Library/YabomishIM/debug.log
```

回報問題時，請附上此日誌檔的相關片段。

### Q：如何完全移除 Yabomish？

執行安裝腳本並選擇移除選項：

```bash
cd yabomish
./yabomish.sh
```

選擇 **6) 移除 Yabomish**。會刪除：

- `/Library/Input Methods/YabomishIM.app`
- `/Applications/YabomishPrefs.app`

移除過程中會詢問是否一併刪除使用者資料（`~/Library/YabomishIM/`）。如果你打算日後重新安裝，建議保留使用者資料以保存字頻學習紀錄。
