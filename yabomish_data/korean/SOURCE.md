# 韓語漢字詞

- **產出**: `terms_korean.bin`
- **來源**: Kengdic（Joe Speigle's Korean/English dictionary）
- **格式**: TSV，欄位：id, surface, hanja, gloss, level, created, source
- **筆數**: 33,414 漢字詞（過濾動詞形 하다/되다/시키다/짓다）
- **授權**: MPL 2.0 / LGPL 2.0+（雙授權，使用者擇一）
- **URL**: https://github.com/garfieldnate/kengdic
- **Build**: `python3 tools/build_korean_hanja.py`
- **注意**: 商業使用選 MPL 2.0，修改的 MPL 檔案需開源（build script 已開源）
