# 台灣地名

- **產出**: `terms_placename.bin`
- **來源**: 教育部「以本土語言標注臺灣地名計畫」
- **格式**: ODT，欄位：號次, 業者代碼, 國語, 臺灣台語, 臺灣客語
- **筆數**: 519 地名（台鐵站名 + 捷運站名）
- **授權**: CC-BY 3.0 TW
- **URL**: https://language.moe.gov.tw/001/Upload/Files/site_content/M0001/mhigeonames/twplacename.html
- **Build**: `python3 tools/build_placename.py`
